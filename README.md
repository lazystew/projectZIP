# projectZIP
A **C++ file archiver implementing custom compression**, archive format design, and lossless decompression, inspired by professional tools like WinRAR.

---

## Core Capabilities

- Custom compression engine implemented in C++
- Entropy-based encoding using Huffman coding
- Bit-level stream packing and unpacking
- Custom archive file format with headers and metadata
- Reliable lossless decompression
- Command-line interface for archive creation and extraction

---

## Planned Extensions

- Multi-file and directory archiving
- Multiple compression modes
- Archive integrity verification
- Streaming compression and extraction
- Performance optimization and profiling

---

## Technology Stack

- **Language:** C++
- **Domain:** Systems Programming
- **Focus:** Compression algorithms, binary formats, performance engineering
